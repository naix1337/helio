Datenmodell: Neue Tabellen

Zwei neue SQLite-Tabellen für Proxmox

📦 proxmox_connections

Speichert die Proxmox-Server-Konfiguration

CREATE TABLE proxmox_connections (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  name            TEXT NOT NULL,
  host            TEXT NOT NULL,        -- IP oder Hostname
  port            INTEGER DEFAULT 8006,
  api_token_id    TEXT NOT NULL,         -- z.B. "root@pam!helio"
  api_token_secret TEXT NOT NULL,        -- verschlüsselt gespeichert
  verify_ssl      INTEGER DEFAULT 0,     -- self-signed Zertifikate erlauben
  poll_interval_s INTEGER DEFAULT 300,   -- 300s = 5 Minuten
  enabled         INTEGER DEFAULT 1,
  last_sync       INTEGER,
  created_at      INTEGER NOT NULL
);

📦 proxmox_resources

Erkannte LXC-Container und QEMU-VMs, gemappt auf Agents

CREATE TABLE proxmox_resources (
  id              INTEGER PRIMARY KEY AUTOINCREMENT,
  connection_id   INTEGER NOT NULL REFERENCES proxmox_connections(id) ON DELETE CASCADE,
  resource_type   TEXT NOT NULL,         -- 'lxc' oder 'qemu'
  vmid            INTEGER NOT NULL,      -- Proxmox-interne ID
  name            TEXT NOT NULL,         -- Container/VM-Name
  agent_id        TEXT REFERENCES agents(id) ON DELETE SET NULL,
  status          TEXT DEFAULT 'running',-- running / stopped / paused
  last_seen       INTEGER NOT NULL,
  created_at      INTEGER NOT NULL,
  UNIQUE(connection_id, resource_type, vmid)
);

↗️ Wichtig: API-Token-Secret wird mit AES-256 verschlüsselt in der DB gespeichert. Der Schlüssel kommt aus einer ENV-Variable HELIO_ENCRYPTION_KEY.
